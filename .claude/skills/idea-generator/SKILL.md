投资想法生成器，支持两种模式：
- `--mode systematic`（默认无 ticker 时）：量化筛选 + 主题扫描，批量发现 5-10 个候选标的
- `--mode narrative`（默认有 ticker 时）：机构级个股深度研报，Goldman 风格完整分析框架

触发词："idea generation", "stock screen", "find ideas", "what looks interesting", "screen for", "new ideas", "pitch me something", "trading idea", "研报", "选股", "找标的", "有什么好的想法"

## Mode 1: Systematic（量化筛选 + 主题研究）

### Step 1: 定义搜索条件

询问用户以下参数（可部分省略，使用默认值）：
- **方向**: 做多 / 做空 / 双向
- **市值**: 大盘 / 中盘 / 小盘 / 微盘
- **板块**: 特定板块或跨板块
- **风格**: 价值 / 成长 / 质量 / 特殊事件 / 事件驱动
- **市场**: 美股 / A股 / 港股 / 全球
- **主题**: 特定主题角度（AI、回流制造、老龄化等）

### Step 2: 量化筛选

根据风格运行对应筛选器：

**价值筛选** — P/E 低于板块中位数、EV/EBITDA 低于历史均值、FCF 收益率 >5%、P/B <1.5x、90 天内有内部人买入、股息率高于市场均值

**成长筛选** — 营收 YoY >15%、盈利 YoY >20%、营收加速（增速递增）、利润率扩张、ROIC >15%、净留存率 >110%（SaaS）

**质量筛选** — 营收连续增长 5 年+、利润率稳定或扩张、ROE >15%、低负债率、高 FCF 转化率、内部人持股 >5%

**做空筛选** — 营收下滑或增速放缓、利润率收缩、应收/存货增长超过销售、内部人卖出、估值溢价无基本面支撑、高空头头寸 + 基本面恶化、会计红旗

**特殊事件筛选** — 近期 IPO/SPAC 锁定期到期、12 个月内分拆、重组中的公司、激进投资者介入、管理层变更

### Step 3: 主题扫描

若有主题角度：
1. 定义论点（如 "AI 基础设施支出加速至 2026"）
2. 映射价值链 — 直接受益者 vs 间接受益者
3. 识别纯正标的 vs 多元化敞口
4. 评估哪些已 "price in" vs 被低估
5. 寻找市场尚未关联到该主题的二阶受益者

### Step 4: 想法呈现

每个通过筛选的标的，输出：

**[公司名称] — [做多/做空] — [一句话论点]**

| 指标 | 数值 | vs 同业 |
|------|------|---------|
| 市值 | | |
| EV/EBITDA (NTM) | | |
| P/E (NTM) | | |
| 营收增速 | | |
| EBITDA 利润率 | | |
| FCF 收益率 | | |

**论点（3-5 点）：** 为何被错误定价、市场遗漏了什么、实现价值的催化剂

**主要风险：** 什么情况下论点会失败

**建议下一步：** 建模？深度尽调？专家访谈？

### Step 5: 汇总输出

- 5-10 个想法的一页摘要
- 筛选标准和方法论记录
- 跨想法对比表
- 优先级排序：哪些想法优先深入研究

## Mode 2: Narrative（机构级深度研报）

### 研究方法

并行执行三路搜索：
1. **财务表现**: 近期财报、营收增长、利润率、关键业务指标、分析师覆盖
2. **市场定位**: 同业对比、板块表现、竞争分析、市场份额
3. **高级情报**: 技术分析、期权资金流、内部人交易、机构持仓、监管风险

### 输出框架

# **[TICKER] — 深度股票研究报告**

## 执行摘要
**[买入/卖出/持有]** 目标价 $[X]（[X]% 上行/下行空间），时间框架 [X] 个月。[核心催化剂与投资论点 1-2 句]。[风险收益比描述]。

## 基本面分析
- **近期财务指标**: 营收增速%、利润率、关键 KPI（含具体数字和时间段）
- **同业对比**: 估值倍数 vs 竞争对手（P/E, P/S, EV/EBITDA）
- **前瞻展望**: 管理层指引、分析师一致预期、增长预测

## 催化剂分析
- **近期（0-6 月）**: 具体事件+日期（财报、产品发布、监管决定）
- **中期（6-24 月）**: 战略举措、市场扩张、竞争格局变化
- **事件驱动**: 并购潜力、指数纳入、分拆、特别股息

## 估值与目标价
当前一致预期 $[X]（区间 $[low]-$[high]）。牛市情景 $[X]（假设...）。基准情景 $[X]（反映...）。熊市情景 $[X]（风险...）。概率权重：[X]%/[Y]%/[Z]%。

## 风险评估
- **公司风险**: 竞争威胁、监管风险、运营挑战
- **宏观风险**: 利率敏感性、经济周期影响、板块轮动
- **仓位建议**: [X]%-[Y]% 配置，基于波动率/Beta/风险因子
- **ESG 考量**: 如对机构持仓有实质影响

## 技术面与期权情报
当前价格 vs 52 周区间。关键支撑/阻力位。近期成交量模式。期权资金流（看跌/看涨比率、异常活动、隐含波动率趋势）。动量指标与趋势分析。

## 市场定位
板块表现（个股 vs 板块 ETF vs 大盘，含具体%）。轮动趋势。相对强度分析。

## 内部人信号
近期内部人买卖（含具体金额和高管姓名）。回购计划。机构持仓变化。内部人行为模式分析。

## 重要提示

- 筛选器产出的是候选名单而非结论 — 每个结果都需要基本面验证
- 最好的想法往往来自交叉区域（如：优质公司因临时逆风出现价值价格）
- 避免拥挤交易 — 检查持仓数据、空头头寸、分析师覆盖密度
- 逆向想法需要催化剂 — 没有催化剂的提前布局等于犯错
- 做空想法需要更高确信度 — 时机更难把握且风险不对称
- 所有财务指标必须包含具体数字和百分比
- 目标价必须展示上行/下行空间计算

**免责声明**: 本分析仅供教育和研究目的，不构成投资建议。过往业绩不保证未来收益。投资决策前请咨询合格金融专业人士。所有投资均存在亏损风险。

## 语言与输出规则

### 中文输出
- 所有分析报告、摘要、建议使用**中文**输出
- 保留英文的情况：ticker 代码（AAPL, 600519.SH）、专有名词首次出现时附英文（市盈率 P/E）、数据源名称
- 数字格式：使用国际通用格式（1,234.56），不使用中文万/亿换算（除非用户要求）

### 多市场 Ticker 识别
根据输入的 ticker 格式自动识别市场并选择数据源：
| 格式 | 示例 | 市场 | 数据源 |
|------|------|------|--------|
| 1-5 位大写字母 | AAPL, MSFT | 美股 | WebSearch / yfinance / FMP |
| 6位数字.SH/.SZ | 600519.SH, 000858.SZ | A 股 | AKShare / 东方财富 |
| 4-5位数字.HK | 00700.HK, 09988.HK | 港股 | AKShare |
| BTC, ETH 等 | BTC, ETH, SOL | 加密货币 | AKShare / CoinGecko |
| XAU, XAG | XAU, GLD | 贵金属 | AKShare |
