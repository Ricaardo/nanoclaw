# AI 投资分析助手

本 skill 提供股票、加密货币、贵金属的实时行情查询、技术指标计算、投资人风格分析等功能。

## 功能

### 1. 实时行情查询

获取股票、加密货币、贵金属、原油的实时价格和历史数据。

**支持的标的**:
- 贵金属: 黄金 (gold)、白银 (silver)
- 原油: WTI原油 (crude_oil)、布伦特原油
- 股指: 标普500 (sp500)、纳斯达克 (nasdaq)、道琼斯 (dow)
- 科技股: NVDA、AAPL、MSFT、GOOGL、AMZN、META、TSLA (MAG7)
- 加密货币: BTC、ETH

### 2. 技术指标

自动计算以下技术指标：
- **RSI** (14日): 相对强弱指数
- **MACD**: 移动平均收敛发散
- **MA**: 移动平均线 (5/20/60日)
- **BB**: 布林带
- **ADX**: 平均趋向指数

### 3. 投资人风格分析

模拟历史上著名投资大师的风格分析市场：

| 投资人 | 风格 | 特点 |
|--------|------|------|
| 巴菲特 | 价值投资 | 长期持有、安全边际 |
| 索罗斯 | 趋势投机 | 顺势而为、果断止损 |
| 芒格 | 逆向投资 | 人弃我取、认知偏差 |
| 达利欧 | 全天候 | 分散配置、风险平价 |

### 4. 持仓管理

- 查看当前持仓和盈亏
- 添加新持仓
- 计算仓位占比

### 5. 交易信号

基于技术分析生成买卖信号：
- 买入信号 (RSI < 30, MACD 金叉)
- 卖出信号 (RSI > 70, MACD 死叉)
- 持有信号

## 触发方式

直接发送以下关键词：

| 关键词 | 功能 |
|--------|------|
| `行情` | 市场概览 |
| `持仓` | 查看持仓和盈亏 |
| `黄金` / `金` | 黄金实时行情 |
| `比特币` / `btc` | 比特币行情 |
| `nvda` / `aapl` / `msft` | 具体股票行情 |
| `巴菲特` | 巴菲特风格分析 |
| `索罗斯` | 索罗斯风格分析 |
| `芒格` | 芒格风格分析 |
| `达利欧` | 达利欧风格分析 |
| `信号` | 交易信号 |

### 示例对话

```
用户: NVDA 现在能买吗?
Agent: [调用 get_market_data 获取 NVDA 行情]
     [调用 get_technical_indicators 计算技术指标]
     [生成分析报告]
     → 返回分析和建议

用户: 用巴菲特的角度分析一下黄金
Agent: [调用 get_market_data 获取黄金数据]
     [调用 get_technical_indicators 计算指标]
     [调用 analyze_investor_style 用巴菲特风格分析]
     → 返回价值投资风格的分析

用户: 我的持仓怎么样?
Agent: [调用 get_portfolio]
     → 返回持仓详情和盈亏

用户: 有什么交易信号?
Agent: [调用 get_trading_signals]
     → 返回各标的的买卖信号
```

## 定时任务

你可以设置定时任务获取每日市场 briefing：

```
用户: 每天早上9点分析市场情况
Agent: [调用 schedule_task 创建定时任务]
     → 每天早上9点自动发送市场分析
```

支持的定时任务：
- 早盘 briefing (9:00)
- 午盘 briefing (13:00)
- 晚盘 briefing (15:00)
- 盘后分析 (16:00)

## 备用机制

如果 yfinance 数据获取失败，系统会自动尝试：
1. WebSearch 搜索实时价格
2. WebFetch 获取财经新闻

## 配置文件

持仓数据存储在: `/workspace/financial-data/portfolio.json`

默认模拟持仓:
- 黄金: 10盎司, 成本 $4800
- 比特币: 0.5枚, 成本 $65000
- 标普500: 100股, 成本 $5800

## 注意事项

1. 本功能仅供学习和参考，不构成投资建议
2. 模拟持仓数据存储在容器内，重建后会重置
3. 如需持久化持仓数据，可挂载持久化存储
